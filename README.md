# VALERIIA — A UNIVERSE MADE OF LOVE

Интерактивный сайт-поздравление для Леры: тёмная романтическая вселенная, сердце из поздравлений на языках мира, 100 причин, библиотека открыток, фотографии, 100 слов, главное письмо, финальная сцена и секретный фиолетовый мишка.

## Быстрая настройка подарка для Леры

1. Открой `src/config/siteConfig.ts` и измени имя, автора, путь к музыке или время старта.
2. Фотографии лежат в `public/photos` с именами `01.jpg`, `02.jpg`, `03.jpg` ... `58.jpg`; новые добавляй следующим номером.
3. Положи музыку в `public/audio/our-song.mp3`.
4. Замени главное поздравление в `src/data/birthdayLetter.ts`.
5. Замени открытки в `src/data/cards.ts` там, где есть комментарии `EDIT HERE`.
6. Запусти сайт командой `npm run dev`.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

После запуска Vite покажет локальный адрес, обычно `http://localhost:5173/`.

На Windows, если PowerShell пишет, что запуск скриптов отключён, используй:

```bash
npm.cmd run dev
```

Или просто открой файл:

```text
START_SITE.cmd
```

## Проверка сборки

```bash
npm run build
```

Собранный сайт появится в папке `dist`.

## Куда добавить фотографии

Фотографии лежат в:

```text
public/photos
```

Используй такие имена:

```text
01.jpg
02.jpg
03.jpg
...
58.jpg
```

Сейчас подключено 58 уникальных фотографий. Если какого-то файла нет, сайт покажет красивую заглушку и не сломается.

Подписи, alt-тексты и позицию кадра можно менять в:

```text
src/data/photos.ts
```

## Как изменить поздравление

Главное письмо находится в:

```text
src/data/birthdayLetter.ts
```

Там есть три версии: `ru`, `en`, `de`. Замени текст внутри `body`.

## Как заменить 100 причин

Причины находятся в:

```text
src/data/reasons.ts
```

Каждая причина имеет русский, английский и немецкий вариант:

```ts
{ ru: "...", en: "...", de: "..." }
```

Сохрани ровно 100 пунктов, чтобы загрузочная сцена и раздел причин работали как задумано.

## Как изменить 100 комплиментов

Комплименты находятся в:

```text
src/data/compliments.ts
```

Русские исходные слова сохранены отдельно в первой колонке массива. Английский и немецкий варианты можно свободно редактировать.

## Как добавить музыку

Музыкальный микшер берёт песни из:

```text
src/data/musicTracks.ts
```

Файлы песен лежат в:

```text
public/audio
```

Сейчас добавлены:

```text
public/audio/until-i-found-you-piano.mp3
public/audio/golden-hour-ruel-remix.mp3
```

Музыка не запускается автоматически. Она стартует только после нажатия кнопки открытия подарка или после ручного нажатия Play в микшере.

## Как изменить время начала музыки

Открой:

```text
src/config/siteConfig.ts
```

Измени:

```ts
musicStartTime: 0
```

Значение указывается в секундах.

## Как изменить имя

Основные настройки находятся в:

```text
src/config/siteConfig.ts
```

Можно менять:

```ts
girlName
fullName
authorName
birthdayDate
musicPath
musicStartTime
defaultLanguage
```

Некоторые художественные фразы с именем Леры дополнительно находятся в `src/data/translations.ts` и `src/data/birthdayLetter.ts`.

## Языки

Сайт поддерживает:

- RU — русский;
- EN — английский;
- DE — немецкий.

Выбранный язык сохраняется в `localStorage`. При первом запуске язык определяется по языку браузера. Если язык не поддерживается, используется русский.

## Открытки и письма

Библиотека чувств находится в:

```text
src/data/cards.ts
```

Можно менять заголовки, тексты, категорию, подпись, фотографию и позицию изображения.

## Развёртывание

### GitHub Pages

1. Выполни `npm run build`.
2. Опубликуй папку `dist` через GitHub Pages.
3. Если сайт будет лежать не в корне домена, измени `base` в `vite.config.ts` на имя репозитория.

### Vercel

1. Импортируй репозиторий в Vercel.
2. Framework Preset: `Vite`.
3. Build Command: `npm run build`.
4. Output Directory: `dist`.

### Netlify

1. Импортируй репозиторий в Netlify.
2. Build Command: `npm run build`.
3. Publish Directory: `dist`.
